**Finder is broken for half the world.** Filaxy is what Finder should have been for power users coming from Windows — dual-pane by default, command palette, duplicate finder, disk usage treemap, bulk rename, Windows-style taskbar, hot-swap ES⇄EN language switcher.

Built solo in 15 days · 80 hours · ~11,000 lines of Swift · **0 external dependencies**.
